Programmes fournis :
valeport_preprocess.py : conversion des fichiers .vp2 en fichiers csv.
TideFile_Convert.py : conversion des fichiers .txt (fichiers de marée récupérés sur datashom.fr) au format .tid (lisibles par CARIS/HIPS)
masque.py : application d'un masque sous forme de .shp à un modèle interpolé avec griddata.
